# Sales Data Analysis Project

## Overview
This project performs end-to-end data analysis on a sales dataset, including data cleaning, preprocessing, filtering, and visualization.


## Objectives
- Clean and prepare raw data
- Handle missing and duplicate values
- Analyze sales trends
- Generate meaningful insights


## Tools & Technologies
- Python
- Pandas
- Matplotlib


## Dataset Features
- Month
- Product
- Category
- Region
- Quantity
- Price
- Discount
- Total Sales


## Steps Performed
1. Data Cleaning (handled missing values & duplicates)
2. Data Preprocessing
3. Data Filtering
4. Feature Engineering (profit, discount %)
5. Data Visualization (line chart, bar chart, stacked chart)


## Key Insights
- Identified the highest sales month
- Found the best-performing product
- Analyzed region-wise revenue
- Observed the impact of discounts on sales


## Visualizations
- Monthly sales trend (Line chart)
- Product-wise sales (Bar chart)
- Category-wise sales (Stacked chart)


## Conclusion
This project helped in understanding the complete data analysis pipeline from raw data to actionable insights.
